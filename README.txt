ISTRUZIONI GITHUB + RENDER

1. Apri GitHub e crea/apri il repository dell invito.
2. Carica questi elementi nella cartella principale del repository:
   - index.html
   - style.css
   - cartella assets intera con page-1.png, page-2.png, page-3.png, page-4.png
3. Su Render crea un nuovo Static Site collegato al repository.
4. Build Command: lascia vuoto.
5. Publish Directory: .
6. Clicca Deploy.

Note: il sito riproduce il PDF come immagini pagina per pagina e mantiene cliccabili i link gia presenti nel PDF: mappa chiesa, mappa ricevimento e RSVP.
